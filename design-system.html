Usage Guidelines

✅ Do

  • Use consistent spacing from the design token scale
  • Follow established color palette
  • Ensure mobile-first responsive design
  • Use translucent button backgrounds (15% opacity)
  • Maintain proper tag vertical centering

❌ Don't

  • Use arbitrary spacing values
  • Create custom colors outside the palette
  • Use JavaScript alerts instead of toast messages
  • Show database IDs in URLs or on screen
  • Implement manual pagination (use infinite scroll)
